# contact-management-app


## 📌 Overview
This is a Contact Management Application built using Python Flask and SQLite.

## 🚀 Features
- Create, Read, Update, Delete contacts
- User-friendly UI
- Data validation
- Persistent storage using SQLite

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## ▶️ Run Project
pip install flask
python app.py

Open: http://127.0.0.1:5000/
